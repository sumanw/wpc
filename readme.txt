== Description ==

This plugin is used for adding authors to Single Post Page.To add authors who have contributed to particular post

*Adds Contributors Section in the post corner right which displays only list of authors
*By Default it displays selected contributors with gravtors at the end of Post 
*To test it on version 5+, please activate gutenberg plugin and then use it to save the post

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wpc` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
